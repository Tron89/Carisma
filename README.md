# Carisma

### ¿Que queremos conseguir con carisma?

Queremos mejorar la interacción de los usuarios, mejorando como comentan, dan like, como ganan amigos y seguidores, y como ganan carisma, puntos virtuales de nuestra aplicacion como el karma de Reddit.

Ademas sentimos que las aplicaciones se han vuelto mas opacas, mostrando solo las buenas reacciones. TikTok solo tiene corazones, YouTube quito el numero de dislikes, y hasta Reddit, no ves realmente cuantos votos negativos tiene un post. (Seremos mas transparentes con las interacciones) Queremos devolver eso, sintiendo que es un sistema necesario, que muestra a los usuarios no solo lo que les gusta, si no tambien saber si ese contenido es útil, permitiendo ver realmente lo que a la gente le gusta y lo que no, permitiendo interacciones mas naturales y menos selectivas y permitiendo contenido mas útil.
